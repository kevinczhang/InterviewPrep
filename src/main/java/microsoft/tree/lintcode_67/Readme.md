## Binary Tree Inorder Traversal

Given a binary tree, return the inorder traversal of its nodes' values.

###Example

**Example 1:**
```
Input：{1,2,3}
Output：[2,1,3]
Explanation:
   1
  / \
 2   3
it will be serialized {1,2,3}
Inorder Traversal
```

**Example 2:**
```
Input：{1,#,2,3}
Output：[1,3,2]
Explanation:
1
 \
  2
 /
3
it will be serialized {1,#,2,3}
Inorder Traversal
```
