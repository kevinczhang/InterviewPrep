## 72. Construct Binary Tree from Inorder and Postorder Traversal

Given inorder and postorder traversal of a tree, construct the binary tree.

###Example
**Example 1:**
```
Input：[],[]
Output：{}
Explanation:
The binary tree is null
```

**Example 2:**
```
Input：[1,2,3],[1,3,2]
Output：{2,1,3}
Explanation:
The binary tree is as follows
  2
 / \
1   3
```
